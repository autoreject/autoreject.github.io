    autoreject.LocalAutoReject¶

    class autoreject.LocalAutoReject(thresh_func=None, consensus_perc=0.1, n_interpolate=0)¶

    Class to automatically reject bad epochs and repair bad trials.

    epochs : instance of mne.Epochs
    The epochs object
    thresh_func : callable | None
    Function which returns the channel-level thresholds. If None, defaults to autoreject.compute_threshes.
    consensus_perc : float (0 to 1.0)
    percentage of channels that must agree as a fraction of the total number of channels.
    n_interpolate : int (default 0)
    Number of channels for which to interpolate
