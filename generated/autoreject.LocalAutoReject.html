    autoreject.LocalAutoReject¶

    class autoreject.LocalAutoReject(thresh_func=None, consensus_perc=0.1, n_interpolate=0, method='bayesian_optimization', verbose='progressbar')¶

    Class to automatically reject bad epochs and repair bad trials.

    epochs
    : instance of mne.Epochs
    The epochs object
    thresh_func
    : callable | None
    Function which returns the channel-level thresholds. If None, defaults to autoreject.compute_thresholds().
    consensus_perc
    : float (0 to 1.0)
    percentage of channels that must agree as a fraction of the total number of channels.
    n_interpolate
    : int (default 0)
    Number of channels for which to interpolate
    verbose
    : ‘tqdm’, ‘tqdm_notebook’, ‘progressbar’ or False
    The verbosity of progress messages. If ‘progressbar’, use mne.utils.ProgressBar. If ‘tqdm’, use tqdm.tqdm. If ‘tqdm_notebook’, use tqdm.tqdm_notebook. If False, suppress all output messages.
