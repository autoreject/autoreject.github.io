    autoreject.compute_thresholds¶

    autoreject.compute_thresholds(epochs, thresh_range=None)¶

    Compute thresholds for each channel.

    epochs : instance of mne.Epochs
    The epochs objects whose thresholds must be computed.
    thresh_range : dict
    Possible keys are ‘eeg’, ‘grad’ and ‘mag’. Each entry is a tuple of the form (low, high) which specifies the range to try.

    For example, we can compute the channel-level thresholds for all the EEG sensors this way:

    >>> compute_thresholds(epochs, range=dict(eeg=(20e-7, 400e-6)))
